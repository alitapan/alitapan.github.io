Quoridor Uygulaması
Quoridor, 2 veya 4 kişilik bir masa üstü strateji oyunudur. Amaç, piyononuzu
oyun tahtasında başladığınız kenardan karşı kenara ilk giden oyuncu olmaktır.
Rakip oyuncuları, oyun tahtasına duvar koyarak engel olabilirsiniz.
Bu uygulama Java dilinde Model-View-Controller (MVC) dizayn paterni kullanılarak
geliştirilmiştir. Grafik kullanıcı arayüzü Java Swing eklentisi ile, uygulamanın
testi Cucumber eklentisi kullanılarak JUnit ve Gradle ile gerçekleştirilmiştir.

leJOS: LEGO Mindstorms için Java
Makina, elektrik ve yazılım mühendisliği alanlarını birleştirerek bir Lego EV3 robotu
tasırımı. leJOS API kullanılarak bu robot ultrasonik sensörle duvarı takip edebilir,
odometre ile kat ettiği mesafeyi ölçebilir, ışık sensörü ile yön bulabilir
ve eğik atış hesapları ile top fırlatabilir.

Tutorolo: Özel Ders Uygulaması
Tutorolo bir web ve Android peer to peer uygulaması olarak öğrenci ve
öğretmeni birleştirmeyi hedefler. Bu uygulama, Java ile Data Object Transfer (DAO)
dizayn paterni ile geliştirilmiştir. Veri tabanı Postgres DBMS ile Heroku aracılığı ile
oluşturulmuştur. Devamlı entagrasyon için Travis CI kullanılmıştır.

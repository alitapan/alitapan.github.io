UNIX: Simple File System
Designed a working, mountable simple file system for Linux.
Limitations:
- Limited length filenames
- Limited length file extensions
- No subdirectories allowed, only a single root directory
- Implemented over an emulated disk system

Quoridor Application
Quoridor is a 2- or 4- player strategic board game developed where the
objective is to be the first player to move their pawn to any space
on the opposite side of the gameboard from which it begins.
This application was developed in Java using the Model-View-Controller (MVC)
design pattern. GUI was developed using Java Swing and testing was done using Cucumber
test runner in conjuntion with JUnit and Gradle build-automation tool.
See more on GitHub
leJOS: Java for LEGO Mindstorms
Combining areas of mechanical, electrical and software engineering, developed an
Lego EV3 robot using the leJOS API to follow walls with an ultrasonic sensor, track
distance traveled with an odometer, navigate using a light sensor and throw objects
using a catapult with projectile motion physics.
See more on GitHub
Tutorolo: Tutoring Application
Tutorolo is a peer-to-peer tutoring web and Android application that aims
to connect students with tutors. It was developed using Data Access Object (DAO)
design pattern. The application was developed using Java. Database was hosted on
a Postgres DBMS instance associated with a Heroku app. A persistence layer was
developed to access data from Backend using Hibernate. Continuous integration
supported by Travis CI.
See more on GitHub
Applied Machine Learning
Projects from graduate level course COMP551 - Applied Machine Learning at McGill University
- Implemention and comparison of two classification techniques, Logistic Regression and
Naive Bayes, on 4 different datasets — These models were built from scratch, without the use of
pre-existing implementations or functions as found in SciKit learn or any other library.
- Development of models to classify textual data, where the input was text documents, and
output were categorical variables (class labels). These models were built using
Logistic Regression, Decision Trees, Support Vector Machines, Ada Boost and Random Forest.
